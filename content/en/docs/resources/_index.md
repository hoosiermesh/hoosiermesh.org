A curated collection of useful resources for Meshtastic users, from official tools to community-created maps and applications.

## Official Meshtastic Resources

### Primary Tools
- [**Meshtastic.org**](https://meshtastic.org) - Official Meshtastic website and documentation
- [**Meshtastic Flasher**](https://flasher.meshtastic.org) - Web-based firmware flasher for all devices
- [**Site Survey Tool**](https://site.meshtastic.org) - Plan your mesh coverage and repeater placement
- [**Meshtastic GitHub**](https://github.com/meshtastic) - Source code, firmware releases, and development

### Documentation & Support
- [**Official Documentation**](https://meshtastic.org/docs/) - Complete technical documentation
- [**Community Discord**](https://discord.gg/ktMAKGBnBs) - Global Meshtastic community chat
- [**Nightly Firmware Builds**](https://nightly.link/meshtastic/firmware/workflows/main_matrix/master) - Bleeding edge builds (advanced users only)

## Network Maps & Visualization

Discover nodes and visualize the mesh network:

- [**MeshMap.net**](https://meshmap.net/) - Global mesh network map with MQTT data and root topic information
- [**Liamcottle's Meshtastic Map**](https://meshtastic.liamcottle.net/) - Alternative global mesh visualization
- [**MeshSense Map**](https://meshsense.affirmatech.com/) - Real-time mesh network monitoring

## Planning & Analysis Tools

- [**RF Line of Sight Tool**](https://www.scadacore.com/tools/rf-path/rf-line-of-sight/) - Calculate RF line-of-sight paths between locations
- [**HeyWhatsThat**](https://www.heywhatsthat.com/) - Panorama and viewshed analysis for antenna placement
- [**Antenna Reports Repository**](https://github.com/meshtastic/antenna-reports) - Community-tested antenna performance data

## Mobile Applications

### Official Apps
- [**Meshtastic Android**](https://play.google.com/store/apps/details?id=com.geeksville.mesh) - Official Android client
- [**Meshtastic iOS**](https://apps.apple.com/us/app/meshtastic/id1586432531) - Official iOS client

### iOS/macOS Apps
- [**MeshScope**](https://apps.apple.com/us/app/meshscope/id6749806329) - Alternative client for Mac and iPad
- [**Mesh World**](https://apps.apple.com/us/app/mesh-world/id6747457936) - Multi-platform mesh client for iPhone, iPad, and Mac
- [**nRF Device Firmware Update**](https://apps.apple.com/us/app/nrf-device-firmware-update/id1624454660) - OTA firmware updates for nRF52 chipsets

### Android Apps
- [**Obtanium**](https://obtainium.imranr.dev/) - Install alpha/beta builds directly from GitHub (advanced users)
- [**nRF Connect**](https://github.com/NordicSemiconductor/Android-nRF-Connect/releases) - OTA firmware updates for nRF52 chipsets
- [**MeshSense**](https://affirmatech.com/meshsense) - Advanced Meshtastic client with additional features

## MQTT & Monitoring

- [**MQTT Explorer**](https://mqtt-explorer.com/) - Monitor MQTT traffic without owning a physical node
- [**mqtt.meshtastic.org**](https://mqtt.meshtastic.org) - Public MQTT server (username: meshdev)

## Educational Resources

- [**CommsChannel YouTube**](https://m.youtube.com/@CommsChannel) - Excellent video guides for setup and configuration
- [**Amateur Radio & Meshtastic**](/docs/reference/ham-radio/) - Information for licensed ham operators
- [**Meshtastic Wiki**](https://meshtastic.org/docs/) - Community-contributed tutorials and guides
